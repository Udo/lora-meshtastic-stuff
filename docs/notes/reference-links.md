## Notes

- Android supports Meshtastic connections over Bluetooth, WiFi on the same network, and USB OTG.
- Meshtastic on ESP32 supports WiFi client mode, not SoftAP mode.
- Enabling WiFi in Meshtastic disables Bluetooth.
